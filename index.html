What is ELEKTRONN?

ELEKTRONN is a toolkit that lifts the predictive power of neural networks on a level accessible to scientists outside the machine learning community.

Powerful & Flexible Training

ELEKTRONN puts emphasis on the analysis of image data using convolutional neural networks (CNN). For this task, we provide a ready-made, flexible training pipeline with helpful utility functions and a comprehensive documentation (+link).

Simple Setup & Optimization

ELEKTRONN is written in Python and based on the Theano framework. On nVidia GPUs if benefits from massive speeups.

About Machine Learning

Many data analysis and classification tasks can be formulated as a machine learning problem: without any explicit expert knowlege or manaul guidance, a powerful generic method — such as a neural network — learns to map certain inputs (e.g. raw images) to certain outputs (e.g. probability maps for classes). The learning is solely driven by a set of exemplary input-output pairs - training examples - that must be provided to the method.
