What is ELEKTRONN?

ELEKTRONN is a deep learning toolkit that makes powerful neural networks accessible to programmers outside the machine learning community. Our focus lies on high troughput analysis of large scale 2d and 3d images with convolutional neural networks (CNNs).

ELEKTRONN can easily installed via pip. Get started

Flexible Training

ELEKTRONN can be used for machine learning tasks formualted on image data or abstract feature vectors. We provide a ready-made, flexible training pipeline with helpful utility functions and comprehensive documentation. See our list of features for more details.

Simple Usage & Optimized Runtimes

ELEKTRONN is written in Python and based on the Theano framework. It can be accelerated with NVIDIA GPUs and delivers state-of-the art performance, including fast training and inference on massive image datasets.

Open Source

ELEKTRONN is an open source project and licensed under the GPL. The code is available at our GitHub repository.

Machine Learning with Neural Networks

Many data analysis and classification tasks can be formulated as a machine learning problem: without explicit expert knowledge or manual guidance artificial Neural Networks can be trained to map certain inputs (e.g. raw images) to outputs (e.g. probability maps for classes). The mapping is found by training on a set of exemplary "input-output pairs"—training examples — that that must be provided initially.
